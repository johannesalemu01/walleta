# Birr Track - Flutter App

A modern expense tracker built for Ethiopian users, written in Flutter.

## Features
- Transaction management (Income & Expense)
- Bank Accounts (CBE, Telebirr, Awash, Dashen, etc.)
- Budget Tracking
- Friends & Loans tracking
- Analytics Dashboard
- App Lock / Security

## Getting Started

To run the Flutter app:

1. Navigate to the Flutter directory:
   ```bash
   cd birr_track_flutter
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
